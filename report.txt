Тестовое задание содержит 5 файлов:
index.php - отображает список сотрудников в таблице
employeehandler.php - содержит класс обработчик. Настройки подключения к бд тоже здесь. 
addingform.php - форма для добавления нового сотрудника
report.txt - файл отчета
employees.sql - дамп базы данных. Содержит 3 таблицы employee, phones и employee_phones(связывает первые 2)

Адрес электронной почты - уникальное значение для каждого сотрудника. Корректность введенного значения проверяется на уровне формы средствами html5(синтаксис) и в классе обработчике (уникальность и необходимость оканчиваться на "@bnb.by")  Проверка на уникальность также реализована на уровне MySQLi

Один и тот же номер телефона может принадлежать нескольким сотрудникам. Несколько номеров отображаются через запятую в индексном файле и вводятся через запятую в поле формы.

В задании используется bootstrap 3. Подключается через https://www.bootstrapcdn.com/
Тестирование проводил на Firefox и Google Chrome 71.0.3578.80 на OpenServer 5.2.2 HTTP:Apache-PHP-7, PHP:PHP-7.1, DB:MySQL-5.6.
